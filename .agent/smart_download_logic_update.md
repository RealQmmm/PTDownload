# ✅ 智能下载逻辑重构与优先级调整

## 变更说明

重新梳理了智能下载的路径匹配优先级，并引入了"下载器默认路径"作为最终兜底选项，替代了原先不确定性较高的"使用首个路径"逻辑。

### 新的优先级逻辑
当进行智能下载时，系统将严格按照以下顺序决定下载路径。如果某个选项被关闭或匹配失败，则进入下一级：

1.  **📂 类型精确匹配** (`match_by_category`)
    *   根据 PT 站点的分类（如"电影"）直接匹配同名的存储路径。
2.  **🔍 关键词评分匹配** (`match_by_keyword`)
    *   根据种子标题中的关键词（如 "S01", "2160p"）与路径名称进行模糊匹配和评分。
3.  **⭐ 系统默认路径** (`fallback_to_default_path`)
    *   使用您在"多路径管理"中标记为 **默认** 的路径。
4.  **📥 下载器默认路径** (`use_downloader_default`)
    *   **新功能**：如果不指定任何路径，直接提交给下载客户端（如 Transmission/qBittorrent），让其使用自身的默认下载目录配置。
    *   此选项可在设置页面开启或关闭。
    *   **注意**：如果此选项也关闭或未命中，自动下载将中止，并提示您手动选择路径。

### 界面变更
*   **设置页面**：移除了 "使用首个路径兜底" 选项，替换为 "**使用下载器默认路径**"。
