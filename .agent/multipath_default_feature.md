# ✅ 多路径管理与默认路径功能上线 (修复版)

## 功能变更

### 1. 📂 菜单更名
侧边栏的 "存储路径" 菜单已更名为 **"多路径管理"**，更准确地反映其功能。

### 2. ⭐ 默认路径标识
在路径管理中，现在可以指定一个 **默认路径**。
- **设置方式**：在添加或编辑路径时，勾选 "设为默认路径 (兜底下载位置)"。
- **UI 展示**：默认路径会在卡片右上角显示 "默认" 标签，并将图标变为金色 ⭐。
- **唯一性**：系统会自动确保任何时刻只有一个默认路径。

### 3. 🤖 智能下载兜底逻辑更新
智能下载的路径推荐逻辑已升级：
1.  **优先级 1**：优先查找带有 `match_by_category` 或 `match_by_keyword` 匹配规则的路径。
2.  **优先级 2**：如果无规则匹配，优先查找**被明确标记为默认**（⭐）的路径。
3.  **优先级 3**：如果未设置默认路径，尝试查找名称为 "其他/默认/default/other" 的路径。

## 故障修复 (2026-01-03 17:35)
